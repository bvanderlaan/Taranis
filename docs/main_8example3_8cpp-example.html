main.example3.cpp

This example shows how to construct a CommandLineInterface object using the static CommandLineInterface::build() method and the CommandLineInterfaceBuilder's fluent interface. Next it shows how to requeste the constructed CommandLineInterface object from the builder. Finally it shows how to access the values the user inputed on the command line.

See Also
main.example1.cpp for a one line way of doing the same thing.

If you run this example when the user provides the mouse flag such as -mouse, –mouse, -m, /m, /mouse You see see the output Show Mouse? Yes otherwise you'll see Show Mouse? No.

#ifdef EXAMPLE3
#include <QCoreApplication>
#include <QDebug>
#include "CommandLineInterface.hpp"
using namespace Taranis;
int main(int argc, char *argv[])
{
QCoreApplication a(argc, argv);
auto builder = CommandLineInterface::build()
.WithName("My Cool App")
.WithVersion("1.2.3.4-abc")
.WithFlag("mouse", "Force mouse to be displayed in release build.");
CommandLineInterface arguments = builder->getCommandLineInterface();
qDebug() << "Show Mouse? " << ( arguments["mouse"].toBool() ? "Yes" : "No" );
return a.exec();
}
#endif
