Taranis::Internal::Argument::callback
std::function< void(QVariant)> callback()
callback is the function you want performed when the argument is detected on the CLI. The call back could be as simple as setting a value to be read by your program later or it could be a handler that performs some action when an argument is present and/or at a given value.
