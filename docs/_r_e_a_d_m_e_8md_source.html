5 Taranis is a toolkit which can be used in Qt projects to provide a command line interface for inputting start up parameters into your application.
6 
7 GUI applications or background processes tend to have flags now and again to allow users to alter how the application will run. They might have
8 flags to change their logging level or alter server connection addresses. The code to process the command line arguments fed into an application
9 is largely boiler plate but to do it nicely is a bit of a pain.
10 
11 Taranis is a class you can instantiate which will handle parsing whatever type of command line argument your application requires. It will deal with
12 case sensitivity and differences in flag syntax such as backslashes (/), dashes (-), or double dashes (--) as well as differences in conventions such as
13 triggering help with either an 'h' or a question mark (?). Taranis will also provide long and short forms of your desired flags such as '--help' or '-h'.
14 
15 Taranis will also take care of generating help text and printing it to the console when the help flag is provided.
